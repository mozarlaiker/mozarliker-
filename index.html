Bem-vindo ao GitHub Pages. Esta página gerador automático é a maneira mais fácil de criar belas páginas para todos os seus projetos. Autor o conteúdo da página daqui usando GitHub Flavored Markdown, selecione um modelo criado por um designer e publicar. Após a sua página é gerada, você pode conferir o novo ramo gh-pages localmente. Se você estiver usando GitHub área de trabalho, basta sincronizar seu repositório e você verá o novo ramo. ### Modelos de Designer Temos trabalhado alguns modelos bonitos para você usar. Vá em frente e clique em "Continuar a layouts 'para navegar por elas. Você pode facilmente voltar a editar sua página antes de publicar. Depois de publicar sua página, você pode revisitar o gerador de página e mudar para outro tema. O conteúdo da página será preservada. ### Criação de páginas manualmente Se preferir não usar o gerador automático, empurrar um ramo chamado gh-pages ao seu repositório para criar uma página manualmente. Além de apoiar o conteúdo HTML regular, Páginas GitHub apoiar Jekyll, um, o blog ciente gerador de site estático simples. Jekyll torna mais fácil para criar cabeçalhos e rodapés em todo o site, sem ter que copiá-los através de cada página. Ele também oferece suporte blogue inteligente e outros recursos avançados de modelagem. ### Autores e colaboradores Pode @mention um nome de usuário GitHub para gerar um link para seu perfil. O <a> elemento resultante vai ligar para GitHub perfil do colaborador. Por exemplo: Em 2007, Chris Wanstrath (@defunkt), PJ Hyett (@pjhyett) e Tom Preston-Werner (@mojombo) fundada GitHub. ### Suporte ou contato Tendo problemas com páginas? Vá para a documentação ou suporte de contato e nós vamos ajudá-lo a resolver o problema.
